# Java Documentation

Java helps us to document our code, for this it provides us with the `javadoc` command.

#### NOTES

- We use the `-d` flag to indicate the folder where the documentation files will be generated.

- The documentation is generated in `html` format, in the folder where the documentation was generated you have to open the `index.html` file so that it can be viewed in the browser.

## Commands used in this section:

To generate javadoc class:
```
javadoc -d docs Calculator.java
```
